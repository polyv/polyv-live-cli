# PolyV 直播 API 业务模块索引

> 本文件是 polyv-live-cli 仓库维护的项目侧索引，不是原始 API 文档。
> API 文档根目录为 `../document-center/docs/live/api`，该路径相对于本仓库根目录解析；例如 `channel/operate/create.md` 对应本仓库同级目录下的 `document-center/docs/live/api/channel/operate/create.md`。

> 本文档基于 614 个 API 文档的实际业务模块分类整理

## 业务模块分类

### 1. 账号管理 (Account Management)
- 用户信息查询
- 分类管理（创建、删除、修改、排序）
- 收入详情
- 使用时长统计
- SSO 单点登录
- 登录令牌管理
- 回调设置（推流、录制、转存回调）
- 客户端启动

### 2. 频道管理 (Channel Management)
- 频道创建（单个/批量/基础/研讨会）
- 频道详情查询

- 频道设置修改
- 频道密码管理
- 频道复制
- 频道删除（单个/批量）
- 频道状态控制（开始/结束直播）
- 频道模板设置
- 频道最大观看人数设置

### 3. 回放管理 (Playback Management)
- 回放列表查询
- 回放开关设置
- 回放视频添加/删除

### 4. 录制管理 (Recording Management)

### 6. 聊天禁言/踢人管理 (Chat Ban Management)

## 建议的 CLI 命令分组

基于以上业务模块分析，建议按以下方式组织 CLI 命令：

| 命令组 | 说明 | 主要功能 |
|--------|------|----------|
| `account` | 账号管理 | 用户信息、分类、SSO、回调设置 |
| `channel` | 频道管理 | 创建、删除、修改、查询频道 |
| `playback` | 回放管理 | 回放列表、开关、视频管理 |
| `record` | 录制管理 | 录制文件、合并、转存 |
| `chat` | 聊天管理 | 消息、禁言、踢人、公告 |
| `product` | 商品管理 | 商品CRUD、排序、统计 |
| `coupon` | 优惠券管理 | 优惠券CRUD、领取记录 |
| `lottery` | 抽奖管理 | 抽奖活动、观众分组 |
| `interaction` | 互动管理 | 签到、问答、问卷 |
| `viewer` | 观众管理 | 观众CRUD、标签、授权 |
| `watch` | 观看条件 | 白名单、外部授权、认证 |
| `stats` | 数据统计 | 观看数据、并发、日志 |
| `player` | 播放器 | Logo、广告、跑马灯 |
| `stream` | 推流管理 | 推流地址、状态控制 |
| `transmit` | 转播管理 | 转播关联、同步 |
| `doc` | 文档管理 | 文档上传、转码、删除 |
| `ppt` | 课件管理 | 课件设置、重制 |
| `donate` | 打赏管理 | 打赏设置、礼物 |
| `distribute` | 分发管理 | 分发频道、统计 |
| `group` | 集团账号 | 分账号、资源分配 |
| `moderation` | 内容审核 | 视频/音频审核 |
| `subtitle` | 字幕管理 | 字幕设置、语言 |
| `ai` | AI 功能 | 数字人、视频制作 |
| `session` | 场次管理 | 场次列表、统计 |
