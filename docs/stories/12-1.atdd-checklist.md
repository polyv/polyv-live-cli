## Expected Interface

### ViewerHandler Class

```typescript
class ViewerHandler extends BaseHandler {
  constructor(authConfig: AuthConfig, serviceConfig: ViewerServiceConfig);

  // AC1
  getViewer(options: ViewerGetOptions): Promise<void>;

  // AC2, AC3
  listViewers(options: ViewerListOptions): Promise<void>;
}
```

### ViewerServiceSdk Class

```typescript
class ViewerServiceSdk {
  constructor(authConfig: AuthConfig, serviceConfig?: ViewerServiceConfig);

  // AC1
  getViewerRecord(params: GetViewerRecordParams): Promise<ViewerRecord>;

  // AC2, AC3
  listViewerRecords(params: ListViewerRecordsParams): Promise<ListViewerRecordsResponse>;
}
```

### CLI Commands

```bash
# viewer get
viewer get -v <viewerId> [-o table|json]
viewer get --viewer-id <viewerId> [--output table|json]

# viewer list
viewer list [--source IMPORT|WX|MOBILE] [--mobile <mobile>]
            [--email <email>] [--area <area>]
            [--page <number>] [--size <number>]
            [-o table|json]
```
