### CLI Commands

```bash
# platform label list
platform label list [-o table|json]

# platform label create
platform label create --name <labelName> [-o table|json]

# platform label update
platform label update --id <labelId> --name <labelName> [-o table|json]

# platform label delete
platform label delete --id <labelId> [-o table|json]
```

## Notes

- Follow the pattern from `PlatformHandler` and `platform.commands.ts`
- Reuse SDK V4UserService methods (listViewerLabels, createViewerLabel, updateViewerLabel, deleteViewerLabel)
- All public methods must have JSDoc comments
- Table output should display labelId, labelName, and createdTime (if available)
- CLI short option rule: **NO `-v` or `-V` allowed** (conflicts with `--version`)
- Use `-o` for output format (consistent with other platform commands)
- Business rule: labelId must be a positive integer
- Business rule: labelName cannot be empty string
