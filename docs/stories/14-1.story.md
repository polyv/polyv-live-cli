# Story 14-1: 营销推广管理命令

## Overview

Implement CLI commands for managing marketing promotion channels (渠道推广). This includes creating, listing, and managing promotional channels for marketing purposes.

## User Story

**As a** 直播运营人员
**I want** 管理营销推广渠道
**So that** 可以追踪不同推广渠道的效果

## Acceptance Criteria

### AC1: `promotion list` - 列出渠道推广列表

- **Given** 用户需要查看频道的所有推广渠道
- **When** 执行 `promotion list --channelId <ID>`
- **Then** 显示所有推广渠道列表，包含推广ID、名称、访问次数、观看人数等统计信息
- **And** 支持 `--output table|json` 参数

### AC2: `promotion create` - 批量创建渠道推广

- **Given** 用户需要为频道创建多个推广渠道
- **When** 执行 `promotion create --channelId <ID> --names <name1,name2,...>`
- **Then** 创建指定的推广渠道并返回创建结果
- **And** 每个渠道名称最大20个字符
- **And** 同一频道最多支持500个推广渠道（包含历史创建）

### AC3: Output Format Support

- **Given** 用户执行任何命令
- **When** 使用 `--output table` 或 `--output json`
- **Then** 以对应格式输出结果
- **And** 默认使用 table 格式

### AC4: 错误处理

- **Given** 用户输入无效参数
- **When** 执行命令
- **Then** 显示友好的错误信息
- **And** 处理 API 错误（401、403、404、500）

### AC5: ATDD 开发模式

- **Given** 开发者遵循 ATDD 流程
- **When** 先编写测试再实现
- **Then** 确保测试先行（RED 阶段）
- **And** 实现使测试通过（GREEN 阶段）

## Technical Notes

### API Reference

- **List API**: `GET /live/v4/channel/popularization/list`
  - 参数: `channelId` (required)
  - 返回: 分页数据包含推广列表

- **Create API**: `POST /live/v4/channel/popularization/create-batch`
  - 参数: `channelId` (required), `names` (required, array)
  - 返回: 创建的推广渠道列表

### Data Model

```typescript
interface PromotionChannel {
  promoteId: string;           // 推广ID
  popularizationName: string;   // 推广渠道名称
  visitsNum: number;          // 访问次数
  reservationNum: number;     // 预约人数
  watchNum: number;           // 观看次数
  viewerNum: number;          // 观看人数
  averageWatchTime: string;  // 人均观看时长
  enrollNum: number;           // 报名观看人数
  createdTime: number;         // 创建时间（13位毫秒时间戳）
}

interface CreatePromotionResult {
  promoteId: string;           // 推广ID
  popularizationName: string;   // 推广渠道名称
}
```

## CLI Commands

```bash
# List promotions
polyv-live-cli promotion list --channelId <ID> [-o table|json]

# Create promotions
polyv-live-cli promotion create --channelId <ID> --names <name1,name2,...> [-o table|json]
```
